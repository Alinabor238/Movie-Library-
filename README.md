## Описание программы (кратко)

GUI‑приложение «Movie Library» для учёта и систематизации фильмов. Позволяет добавлять, просматривать и фильтровать информацию о фильмах, а также сохранять данные в формате JSON для долгосрочного хранения. Программа подходит для создания персональной базы фильмов с возможностью быстрого поиска по ключевым параметрам.

### Возможности (что программа умеет делать)

* **Добавление фильмов** через удобную форму с полями:
    * **Название** — обязательное текстовое поле;
    * **Жанр** — обязательное поле, можно выбрать из списка или ввести новый жанр;
    * **Год выпуска** — числовое поле (проверяется корректность ввода);
    * **Рейтинг** — число от 0 до 10 (с проверкой диапазона).
* **Отображение коллекции** в виде интерактивной таблицы с возможностью просмотра всех добавленных фильмов.
* **Фильтрация данных**:
    * по жанру (выпадающий список с вариантами);
    * по году выпуска (точный год).
* **Сохранение данных** в файл `movies.json` при добавлении нового фильма или закрытии приложения.
* **Загрузка данных** из файла `movies.json` при запуске программы. Если файл отсутствует, создаётся новый.
* **Валидация данных**:
    * проверка, что год выпуска — целое число в разумном диапазоне (например, 1888–2030);
    * проверка рейтинга: значение должно быть числом от 0 до 10 включительно (допускаются дробные значения);
    * обязательность заполнения полей «Название» и «Жанр».
* **Интерфейс** с интуитивно понятными элементами управления и мгновенной обратной связью при ошибках ввода.

## Инструкция по запуску
Запустите программу:

Для Windows:

python main.py
Для macOS/Linux:

python3 main.py

### Требования для запуска программы

* **Python** версии 3.8 или выше.
* **Установленные библиотеки**:
    * `tkinter` — стандартная библиотека Python для создания графического интерфейса (обычно входит в поставку);
    * `json` — встроенная библиотека Python для работы с JSON‑файлами;
    * `os` — для работы с файловой системой (встроенная).
* **Операционная система**: Windows, macOS или Linux.

Примеры использования программы
Пример 1. Добавление фильма

Запустите программу командой python main.py или python3 main.py.

Заполните форму добавления фильма:

Название: «Интерстеллар»;

Жанр: «фантастика» (выберите из списка или введите вручную);

Год выпуска: 2014;

Рейтинг: 8,6.

Нажмите кнопку «Добавить фильм».

Проверьте таблицу ниже — новый фильм должен появиться в списке.

Если вы ввели некорректные данные (например, год «abc» или рейтинг 15), появится сообщение об ошибке с указанием, что нужно исправить.

Пример 2. Фильтрация по жанру

В выпадающем списке «Фильтр по жанру» выберите нужный жанр, например, «драма».

Нажмите кнопку «Применить фильтр».

Таблица обновится и покажет только фильмы жанра «драма».

Чтобы сбросить фильтр, выберите в списке «Все жанры» и нажмите «Применить фильтр» — отобразятся все фильмы.

Пример 3. Фильтрация по году

Введите в поле «Фильтр по году» нужный год, например, 2019.

Нажмите кнопку «Применить фильтр».

Таблица покажет фильмы, выпущенные в 2019 году.

Для сброса фильтра очистите поле или введите 0 и нажмите «Применить фильтр» — вернутся все записи.

Пример 4. Сохранение и загрузка данных

Добавьте несколько фильмов через форму.

Закройте программу — данные автоматически сохранятся в файл movies.json.

Снова запустите программу командой python main.py или python3 main.py — ранее добавленные фильмы загрузятся из movies.json и отобразятся в таблице.

Если вы удалите файл movies.json и перезапустите программу, она создаст новый пустой файл.

Пример 5. Обработка ошибок ввода

Попробуйте добавить фильм с некорректными данными:

год выпуска: «двадцать двадцать четыре» → появится ошибка «Год должен быть числом»;

рейтинг: 12 → появится ошибка «Рейтинг должен быть от 0 до 10»;

пустое поле «Название» → появится ошибка «Заполните все обязательные поля».

Исправьте ошибки и повторите попытку — фильм добавится успешно.
